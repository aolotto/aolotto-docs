AO是继比特币，以太坊之后颇具革命性的范式创新，基于AO构建的去中心化应用具有无限的扩展性和永久的访问性，用户通过AO发起链上交易无需承担高昂的GAS消耗，这为Aolotto提供了足以媲美甚至超越Web2.0的使用体验，用户无需排队交易，无需承受网络拥堵之苦。
